# tiny-toolkit-pro

## 说明

tiny-toolkit-pro 套件，开箱即用的中后台前端/设计解决方案

## 用法

### 初始化

```
tiny init pro
```

### 配置项目名称、描述、技术栈

```
 ? 请输入项目名称： 
 ? 请输入项目描述： (基于TinyPro套件创建的中后台系统)
 ? 请选择您希望使用的技术栈： (Use arrow keys)
 > vue
   angular
```

### 开启本地服务器

```
tiny start
```

### 代码打包

```
tiny build
```
